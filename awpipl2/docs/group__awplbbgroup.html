receives LBP code for the smeared point of the image.

Parameters

[in]-pis the point on the image for which the code is to be calculated.
[in]-shiftX- horizontal offset of the point
[in]-shiftY- point offset vertically.
[in]-scale- the coefficient. stretching the neighborhood. (> = 1)
[in]-options- a parameter indicating which variation of the algorithm to use for
get the code. Must be one of the AWP_LBP_XXX constants.
AWP_LBP_U1R1 - nonuniform transformation of radius 1.
AWP_LBP_U2R1 - uniform transformation of radius 1.
ununiform codes - * pcode = 0
AWP_LBP_U1R2 - nonuniform transformation of radius 2.
AWP_LBP_U2R2 - uniform transformation of radius 2.
nonuniform codes * pcode = 0
