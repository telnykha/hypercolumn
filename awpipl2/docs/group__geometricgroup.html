AWPRESULT awpBilinearInterpolation ( awpImage *  src,
awp2DPoint *  p,
awpImage *  dst 
)

find value of bilinear interpolation in the souce image with subpixel accuracy

Basic AWPIPL functions for geometric transformation. It defines functions for flipping, resizing, rescaling and rotation of AWPImage.

Parameters
awpImage*src - source image
awp2DPoint*p - point where the value sould be calculated
awpImage*dst - interpolated value, should be awpImage type AWP_DOUBLE and size 1x1
Returns
AWP_OK if success or else AWP_ERROR.
